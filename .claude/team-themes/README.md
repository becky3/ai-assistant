# チームテーマ定義

このディレクトリには、エージェントチームのキャラクターテーマ定義ファイルを格納します。

## 概要

「チームで開発して」という指示があった場合、このディレクトリからランダムにテーマが選択され、
そのテーマのキャラクターたちがチームメンバーとして開発を進めます。

## ファイル構成

各テーマは以下の形式で定義されます:

```
{theme-name}.md
```

## テーマファイルの構成

```markdown
# テーマ名

## 作品設定

作品の世界観、背景設定を記載

## チームメンバー

| ロール | キャラクター名 | 性格・特徴 | 担当 |
|--------|--------------|-----------|------|
| リーダー | キャラA | 性格特徴 | 全体調整 |
| メンバー1 | キャラB | 性格特徴 | 仕様担当 |
| メンバー2 | キャラC | 性格特徴 | 実装担当 |
| メンバー3 | キャラD | 性格特徴 | 品質担当 |

## 目標の意味付け

開発タスクを作品の物語に沿った形で解釈する方法

## 会話スタイル

キャラクターらしい会話や行動の指針
```

## 使用方法

1. ユーザーが「チームで開発して」と指示
2. システムがこのディレクトリからテーマをランダム選択
3. 選択されたテーマのキャラクター設定でチームを構築
4. キャラクターの性格・特徴に基づいた自然な会話で開発を進行
